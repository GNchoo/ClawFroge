# ClawForge Backstory

## The Creators
The creators of ClawForge were featured in the Wall Street Journal for their innovative approach to AI coding assistants. They are avid fans of Claude Code and wanted to create a "Better Harness Tool" that actually gets things done.

## Why this rewrite exists
The original Claude Code is a powerful CLI tool, but it has limitations in terms of accessibility and multi-model support. ClawForge aims to:
- Provide a high-performance web interface.
- Support both cloud-based (Gemini) and local LLMs.
- Implement an autonomous agent harness that can manipulate the file system directly.
- Use Rust and `oh-my-codex` for high-speed execution and reliability.
