⭐ The fastest repo in history to surpass 50K stars, reaching the milestone in just 2 hours after publication ⭐

## Overview
Better Harness Tools, not merely storing the archive of leaked Claude Code. Now writing in Rust using oh-my-codex.

## Philosophy
- **Autonomous Execution**: The AI should be able to navigate the codebase, read files, and apply changes with minimal user intervention.
- **High-Speed Search**: Instant access to code patterns across the entire project.
- **Multi-Model Support**: Seamlessly switch between high-performance cloud models (Gemini) and privacy-focused local LLMs.
- **Terminal Integration**: Real-time feedback of AI actions and tool executions.

## Repository Layout
- `src/`: Core logic and UI components.
- `server.ts`: Backend API for file system and search operations.
- `docs/`: Project documentation and guides.
