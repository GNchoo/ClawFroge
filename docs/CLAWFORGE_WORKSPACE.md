# ClawForge Workspace Layout

## Python Workspace Overview
The workspace is designed to be highly efficient for AI agents. It follows a modular structure that allows the AI to quickly locate and modify specific parts of the codebase.

## Quickstart
1. **Initialize the Harness**: Start the ClawForge server.
2. **Select a Model**: Choose between Gemini or a local LLM.
3. **Execute Tools**: Use the `list_files`, `read_file`, `write_file`, and `search_code` tools to interact with the project.
4. **Monitor Progress**: Watch the real-time terminal for tool execution logs.
